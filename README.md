#  API Blog - Projet INF222

Ce projet est une API Backend développée avec **Node.js** et **Express**, utilisant une base de données **SQLite** pour la gestion d'un blog (articles).

## Fonctionnalités
- Gestion complète des articles (CRUD : Create, Read, Update, Delete).
- Documentation de l'API avec **Swagger (OpenAPI 3.0)**.
- Base de données légère et intégrée avec **SQLite**.

## Technologies utilisées
- **Backend** : Node.js, Express.js
- **Base de données** : SQLite3
- **Documentation** : Swagger-jsdoc & Swagger-ui-express
- **Environnement** : Ubuntu Linux / VS Code

##  Structure du Projet
- `controllers/` : Logique métier (articleControler.js).
- `models/` : Schémas et gestion des données.
- `routes/` : Définition des points d'entrée (articleRoutes.js).
- `server.js` : Point d'entrée principal de l'application.
